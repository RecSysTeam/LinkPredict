# LinkPredict

## Instalation 
```sh
git clone --recursive https://github.com/RecSysTeam/LinkPredict.git
```
```sh
cd LinkPredict
```
```sh
python3.9 -m venv .rec_sys_proj_venv
```
```
source .rec_sys_proj_venv/bin/activate
```
```sh
pip install -r requirements.txt
```
For contributions
```sh
pip install pre-commit
```
